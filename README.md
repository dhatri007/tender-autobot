# 🚀 AutoTender AI
## Smart Tender Analysis & Bid Optimization System

---

## 🧠 What is AutoTender AI?

**AutoTender AI** helps companies prepare **winning tender bids automatically**.

Instead of manually analyzing tender documents and calculating prices manually, the system:

✅ Reads tender files  
✅ Matches best products automatically  
✅ Suggests optimized bid pricing  
✅ Calculates winning probability  
✅ Generates a professional tender proposal (PDF)

---

## ✨ Key Features

### 📄 Tender Upload
Upload a tender document directly from your computer.

➡️ The system analyzes the file and generates a clear **Tender Summary** automatically.

---

### 📦 Product Catalog
Add products using:

- CSV Upload
- Manual Product Entry

Each product contains full technical and pricing information such as:

- Name
- Type
- Finish
- VOC
- Coverage
- Price per litre
- Packaging
- Location

---

### 🔎 Best Match & Gap Analysis
AutoTender AI intelligently finds the most suitable products for the tender.

Displays:

- Match score (0–1 scale)
- Requirement gaps
- Suggestions to improve compatibility
- Technical comparison insights

---

### 💰 Bid Recommendation (Core Intelligence)
Smart pricing engine generates an optimized bid table including:

- Discount per litre
- Bid price per litre
- Total bid value
- Profit percentage
- Winning probability

Users can adjust **profit threshold** to explore better bidding strategies.

---

### 🏢 Company Details → Final Proposal
Enter company information and generate a:

📑 **Professional Final Tender Proposal PDF**

The PDF includes:

- Company details
- Tender summary
- Selected product specifications
- Commercial bid table
- Final pricing details

---

### 📊 History Dashboard
Tracks all generated tenders with:

- Tender ID
- Status
- Bid value
- Generated proposal history

---

## ⚙️ Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- React Router
- Context API (State Management)
- jsPDF + AutoTable (PDF Generation)

## 🧪 How to Run the Project

```bash
git clone <repo-url>
cd autotender-ai
npm install
npm run dev
```

Open in browser:

```
http://localhost:5173
```
